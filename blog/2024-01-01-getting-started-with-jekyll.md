Jekyll is a static site generator that works great with GitHub Pages. Here's how to get started.

## Why Jekyll?

- Simple markdown-based content
- Free hosting with GitHub Pages
- Fast performance
- Easy to maintain

## Quick Setup

```bash
gem install jekyll bundler
jekyll new site-name
cd site-name
bundle exec jekyll serve
```

Your site will be available at `http://localhost:4000`.

## Customization

Edit `_config.yml` to update your site settings, and start creating content with markdown files. Happy coding!
